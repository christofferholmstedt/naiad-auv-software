Package: Vectors
  • Goto spec source

Description

package Math.Vectors is
Vectors package for classes, types and functionality regarding vectors. A vector object is stored in a pCVector variable and is created with the pxCreate function.

type pCVector is access CVector;
Pointer type for object of type CVector. Objects of type CVector should always be stored in variables of this type.

pxCreate
  • Goto spec in source
 

function pxCreate 
(fX, fY, fZ: float) return pCVector;
Creates an object of type CVector. Returns a pointer of type pCVector to the object created.
fX: The value of the vector's X-component.
fY: The value of the vector's Y-component.
fZ: The value of the vector's Z-component.

pxGet_Copy
  • Goto spec in source
 

function pxGet_Copy 
(this: in CVector) return pCVector;
Creates an object of type CVector with the same component values as the object called on. Returns a pointer of type pCVector to the object created.
this: The CVector object to copy the component values from.

pxGet_Normalized

function pxGet_Normalized 
(this: in CVector) return pCVector;
Creates an object of type CVector which is equal to the normalized vector of the object called on. Returns a pointer of type pCVector to the object created.
this: The CVector object to base the normalized vector on.
