Package: Planes

Description

package Math.Planes is
Planes package for classes and functionality regarding planes. A plane object is stored in a pCPlane variable and is created with the pxCreate function.

function pxCreate 
(pxNormalVector: in Math.Vectors.pCVector;
fDistanceFromOrigin: in float) return pCPlane;
Creates an object of type CPlane. Returns a pointer of type pCPlane to the object created.
pxNormalVector: The normal vector of the plane to be created.
fDistanceFromOrigin: The distance from origin to plane along the normal vector.
Numeric_Error, if pxNormalVector vector length = 0.0.
